FRENDS⁴ Common Components : Frends.Common.Files.Operations.ReadFile

Created by Emma Luukka, last modified by Frends Editor on Mar 21, 2016

This task reads the contents of a file from a given location.

Input Parameters

  • FilePath
    • The path to the file that is to be read.
  • Encoding
    • The encoding of the source file. Different encoding supported can be found here https://msdn.microsoft.com/en-us/library/windows/desktop/dd317756%28v=vs.85%29.aspx

Output Result

  • String
    • Returns the contents of the file
