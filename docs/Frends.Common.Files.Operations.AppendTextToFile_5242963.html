FRENDS⁴ Common Components : Frends.Common.Files.Operations.AppendTextToFile

Created by Emma Luukka, last modified by Frends Editor on Mar 21, 2016

This task appends the input to an existing file or creates a new file if one already exists.

Input Parameters

  • Input
    • The path to the file that the input will be appended into.
  • FilePath
    • The data that will be appended to the file.
  • Encoding
    • The encoding of the destination file. Different encoding supported can be found here https://msdn.microsoft.com/en-us/library/windows/desktop/dd317756%28v=vs.85%29.aspx

Output Result

  • String
    • Returns the path of the appended file
