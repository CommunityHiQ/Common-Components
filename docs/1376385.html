Frends.Common is set of pre-built FRENDS⁴ tasks, that is free for download and usage with FRENDS⁴. The aim for these tasks is to provide the basic tools, in the form of FRENDS⁴ tasks, to start building processes and services with FRENDS⁴.

This wiki contains

  • Information about the tasks
  • Usage of the tasks
  • Things to note when using the common tasks
  • Installation and update guide

You can download the latest versions of the Common -tasks on our Bitbucket page.

For more information about FRENDS⁴ itself visit our Website or our Youtube -channel.

Common Task Packages

The common tasks are divided into six different packages, each with similar functionality. The "Component specific documentation" section is also divided into using the same logic:

  • Frends.Common.Data
  • Frends.Common.Database
  • Frends.Common.DelayProcess
  • Frends.Common.Execute
  • Frends.Common.Files
  • Frends.Common.Services

Installing and Updating Tasks

All FRENDS tasks are installed using Nuget -packages, which can be downloaded straight from this BitBucket page. To install new tasks simply import the Nuget -package to FRENDS as follows:

  1. Click the Import FRENDS NuGet button on the Tasks view

  1. Navigate to the NuGet -package you wish to add and select "Open"

  2. Confirm you want to add the NuGet -package
