FRENDS⁴ Common Components : Frends.Common.Files.Operations.MoveFiles

Created by Frends Editor, last modified on Mar 21, 2016

This task moves files from a source path to the target path.

Input Parameters

  • SourcePath
    • The path to the file that is to be moved
  • FileMask
    • File mask used to filter files
  • TargetPath
    • The destination path where the file is being moved to
  • Overwrite
    • Can be used to overwrite destination file if it exists

Output Result

  • String
    • Returns the list of files moved
