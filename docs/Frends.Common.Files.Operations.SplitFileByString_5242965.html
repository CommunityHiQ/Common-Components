FRENDS⁴ Common Components : Frends.Common.Files.Operations.SplitFileByString

Created by Emma Luukka, last modified by Frends Editor on Mar 21, 2016

This task splits an existing file by a given input string and returns the split file contents as a list of strings.

Input Parameters

  • FilePath
    • The path to the file that will be split
  • SplitString
    • The string that will be used to split the file
  • Encoding
    • The encoding of the source file. Different encoding supported can be found here https://msdn.microsoft.com/en-us/library/windows/desktop/dd317756%28v=vs.85%29.aspx

Output Result

  • List
    • Returns a list of the strings of the split
