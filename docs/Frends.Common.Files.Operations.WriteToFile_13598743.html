FRENDS⁴ Common Components : Frends.Common.Files.Operations.WriteToFile

Created by Frends Editor, last modified on Mar 21, 2016

This task writes input to destination file. File will be created if it doesn't exist. If it exists overwrite flag can be used to overwrite it.

Input Parameters

  • Input
    • String to write to file.
  • DestinationFolder
    • The path to the file that is to be written to.
  • Encoding
    • The encoding of the destination file. Different encoding supported can be found here https://msdn.microsoft.com/en-us/library/windows/desktop/dd317756%28v=vs.85%29.aspx
  • Overwrite
    • Overwrite existing file

Output Result

  • String
    • Returns the contents of the file
